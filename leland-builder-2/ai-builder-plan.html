What was built
Created the AI Automation Roadmap database in Notion with fields for Name, Description, Priority, Status, Submitted By, and Submitted At. Built a serverless API (api/roadmap.js) deployed to Vercel with GET (fetch all rows) and PATCH (update Status by page ID) endpoints.

The API reads NOTION_TOKEN and NOTION_DATABASE_ID from Vercel environment variables — credentials never touch the frontend. Notion Status values map cleanly to board columns (e.g. "In Progress" → in-progress), so the board always reflects the real state of the roadmap.

A daily n8n workflow that reads your inbox, filters noise, and drafts replies in your voice — with a Slack summary and zero sends without your approval.

How It Runs
The workflow fires every day at 4pm on a schedule trigger. It pulls all emails from the last 24 hours via Gmail OAuth2, extracts key fields (sender, subject, body), and strips signatures, footers, and thread noise to reduce token overhead before anything hits Claude.
Categorization Logic
Each cleaned email is passed individually to Claude Haiku — fast and cheap — with a system prompt that classifies it into one of four buckets: URGENT, REPLY_NEEDED, FYI, or NOISE. Haiku returns a structured JSON object with the category and a one-sentence reason.

Only URGENT and REPLY_NEEDED emails pass the If-branch. FYI and NOISE are dropped before any drafting work happens.
What Triggers a Draft
Emails that clear the Haiku filter go to the Claude Sonnet drafter agent, which has access to a Gmail Draft tool. It reads the original email, applies your voice and context (role at 47G, tone, typical reply patterns), and creates a draft in Gmail — not sends it. Nothing leaves without you reviewing and clicking send yourself.

After all drafts are created, a Summarize node compiles the list and the workflow fires a Slack DM to you with: how many emails were processed, how many drafts were created, and direct links to the urgent ones.

How to Deploy
  1. Create an n8n Cloud account and start a new workflow.
  2. Add a Schedule Trigger node — set to daily at 4pm local time.
  3. Connect Gmail OAuth2 and add a Get Many Messages node filtered to the last 24 hours.
  4. Add a Set node to extract key fields (from, subject, body snippet) and an Edit Fields or Code node to strip signatures and thread clutter.
  5. Add a Basic LLM Chain node connected to the Anthropic Chat Model (Haiku). Paste the Email Categorizer system prompt below. Parse the JSON output.
  6. Add an If node: pass through only emails where category equals URGENT or REPLY_NEEDED.
  7. Add an AI Agent node connected to Anthropic Chat Model (Sonnet). Paste the Email Drafter system prompt below. Give the agent access to the Gmail: Create Draft tool.
  8. Add a Summarize / Code node to aggregate draft results into a message string.
  9. Add a Slack node (OAuth2) to send a DM to yourself with the summary and any urgent email links.
  10. Activate the workflow. No laptop required — it runs on n8n's cloud infrastructure.
Email Categorizer System Prompt (Haiku)
You are an email triage assistant for Gerritt Vander Linden, Senior Director of Partnerships at 47G — Utah's aerospace, defense, and hard tech association. Classify the email below into exactly one category: - URGENT: Requires immediate action or a response today. Time-sensitive requests, anything from a board member, partner, or member with a hard deadline, event logistics in the next 48 hours. - REPLY_NEEDED: Requires a response but not time-sensitive. Follow-ups, introductions, partnership inquiries, membership questions. - FYI: Informational only. No action required. Event recaps, newsletters Gerritt actually reads, org updates, meeting notes. - NOISE: No value. Marketing emails, automated notifications, receipts, LinkedIn digests, calendar spam, re-engagement sequences. Return ONLY a JSON object in this exact format — no other text: {"category": "URGENT" | "REPLY_NEEDED" | "FYI" | "NOISE", "reason": "One sentence explaining your classification."}
Email Drafter Agent System Prompt (Sonnet)
You are Gerritt Vander Linden's email drafting assistant. Gerritt is Senior Director of Partnerships & Global Initiatives at 47G — Utah's aerospace, defense, and hard tech association. Your job: read an incoming email and write a reply draft that sounds exactly like Gerritt. Save it as a Gmail draft using the Gmail Create Draft tool. Do not send it. Voice and tone: - Casual and direct. No stiff corporate language. - Short paragraphs, occasionally a single sentence reply. - Warm but efficient — Gerritt values other people's time. - Uses "Happy to" and "Let me know" naturally. - Signs off with "— Gerritt" or just "Gerritt" - Does not use phrases like "I hope this email finds you well" or "Please don't hesitate to reach out." Context about Gerritt's work: - Manages partnerships, sponsorships, and membership for 47G. - Frequently coordinates events, delegation visits, and member introductions. - Works with aerospace and defense companies, government agencies, universities, and investors in Utah and internationally. - His top goal is 35% YoY revenue growth through memberships and partnerships. Draft guidelines: - Match the length and formality to the original email. A short email gets a short reply. - If the email is an intro or partnership inquiry, be warm and suggest a brief call. - If the email is a member request, route it appropriately or confirm receipt and next steps. - If the email is urgent, draft a response that acknowledges the urgency and provides a clear next step. - Do not make up facts. If you need specific details (dates, prices, names), use a clear placeholder like [DATE] or [CONFIRM THIS]. - Always create the draft in Gmail using the tool — do not just return text.
